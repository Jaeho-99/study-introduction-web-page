1. 연구 배경 및 아이디어

1.1. 연구 동기

위 기사의 헤드라인과 같이 자율 주행과 같은 교통 서비스를 위해서는 정밀한 위치 기반 서비스가 필요합니다. 이러한 위치 기반 서비스를 통해서 도로 네트워크를 예측할 수 있으며, 이를 통하여 내비게이션 시스템, 교통 계획 애플리케이션 등과 같은 다양한 교통 서비스를 유용하게 활용할 수 있습니다. 이와 같은 위치 기반 서비스를 위해서는 정밀한 맵 매칭 기술이 필요합니다. 그리고 현재까지 사용되는 맵 매칭 모델 중 가장 보편적인 모델인 Hidden Markov Model(HMM) 기반 맵 매칭 방법이 있습니다. 하지만 기존의 HMM의 경우, 현재 상태가 그 이전 상태에만 의존한다는 Markov chain rule을 따르기에 시계열 데이터의 종속성을 과하게 단순화시킨다는 문제점이 있습니다. 이로 인해, 도심지에서 볼 수 있는 복잡한 도로 관계 및 이동 패턴을 보이는 경우나, 관측치 에러 및 데이터 샘플링 간격이 큰 경우를 접할 때 잘못된 맵 매칭 결과를 보여줍니다.
따라서, 위와 같은 기존의 HMM 방법의 단점을 보완하기 위해서 본 연구를 진행하였습니다.

1.2. 단어 안내 웹 사이트

  • Hidden Markov Model(은닉 마코프 모델)
  • Map Matching(맵 매칭)

1.3. 아이디어

기존의 HMM 모델에서 제기되었던 데이터 간의 종속성 단순화 관련 문제점을 해결하기 위한 방법을 모색하였습니다. 기존의 HMM 모델의 경우 현재 상태(t 시각)의 데이터는 그 이전 상태(t-1 시각)의 데이터만 고려한다는 문제점에 대해서, 더 넓은 범위의 시간 종속성을 고려하도록 알고리즘을 수정하면 문제를 보완할 수 있다고 생각하였습니다. 따라서, 만약 t 시각의 위치를 예측한다고 가정하였을 때, 기존에는 t-1 시각의 데이터에만 의존하였지만, 새로운 방법으로, 데이터를 여러 개로 묶어 {t-1, t-2, t-n, t+1, t+2, t+n}의 데이터에 대해서 가장 이동 방향 경향성(trend)를 잘따르는 데이터로 맵 매칭을 적용하도록 알고리즘을 수정해보기로 하였습니다.

2. 연구 계획

  • 1. 자료조사 : 관련 자료(논문 및 기술 등) 탐색 및 정리.
    - HMM 기술을 기반으로 한 맵 매칭 알고리듬 논문자료를 탐색하여 관련되는 기술 및 정확도 등을 정리합니다.

  • 2. 데이터 수집 : 실험에 사용할 데이터를 수집.
    - Dataset for testing and training map-matching methods에서 맵 매칭 알고리즘에 사용할 수 있는 데이터를 다운로드 받습니다. 해당 데이터셋은 맵 매칭 알고리즘을 위한 데이터셋으로, 세계 여러 지역에서의 차량 이동 데이터로 구성되어 있습니다.

  • 3. 데이터 전처리 : 실험을 위한 데이터 전처리.
    - 이동 데이터의 데이터 전처리 알고리즘인 Ramer-Douglas-Peucker algorithm을 적용한 데이터셋을 추출합니다. 그리하여 데이터 전처리 알고리즘이 적용된 데이터셋과 raw 데이터셋을 준비합니다.

  • 4. 모델 구조 디자인 : 자료를 기반으로 한 맵 매칭 모델 디자인.
    - 필요한 함수 및 클래스 등을 구성하며 전반적인 모델의 구조를 디자인합니다.

  • 5. 모델 구현 : 디자인된 각 요소들의 구현.
    - 모델 구조 디자인을 기반으로 하여 각 요소들을 구현(프로그래밍)합니다.

  • 6. 모델 테스트 및 보수 : 구현 완료한 모델 테스트 및 보수.
    - 구현 완료한 각 요소들을 통합하여 모델을 테스트합니다. 테스트 과정을 거친 후, 결과를 기반으로 모델을 보수합니다.

  • 7. 성능 비교 : 기존의 HMM 모델과 새 모델 trendHMM 모델의 결과값 비교.
    - 최종적으로 완성된 모델을 가지고 기존의 HMM 모델과 새 모델 trendHMM 모델의 결과값을 가지고 새 모델의 성능을 평가합니다. 성능 평가의 지표는 기존의 HMM 논문(Hidden Markov map matching through noise and sparseness)에서 사용되었던 Route Mismatch Fraction(RMF, 경로 불일치 비율)로 선택합니다.

  • 8. 논문 작성 : 실험 결과를 기반으로 한 논문 작성.
    실험 결과를 기반으로 논문을 작성합니다. 기존 모델과 새로운 모델과의 기본 성능 차이, 데이터 전처리 과정에 따른 성능 차이를 근거로 하여 새로운 모델의 우수성을 기준으로 논문을 작성합니다.

  • 9. 최종 검토 : 논문 투고 및 수정.
    작성 완료된 논문을 해외 저널에 투고한 후, 수정 과정을 수행합니다.
